# Notes App

A simple and clean Notes App built with HTML, CSS, and JavaScript.
Users can create, edit, and delete notes through an easy interface.

## Features

* Add new notes
* Edit existing notes
* Delete notes
* Minimal and user-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

notes-app/
│
├── index.html
├── style.css
├── main.js
├── images/ (if used)

## How to Use

1. Download or clone the repository
2. Open `index.html` in your browser
